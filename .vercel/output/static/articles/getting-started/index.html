开始使用知识库

快速了解如何使用这个前端知识库系统

2026/4/6 · 5 分钟阅读

欢迎

这是你的第一篇知识库文章。你可以使用 MDX 语法来编写内容，支持 React 组件。

特性

  • Astro - 高性能静态站点生成
  • React 岛屿 - 按需水合交互组件
  • Tailwind CSS - 实用优先的样式
  • Pagefind - 全文搜索

编写文章

在 src/content/articles/ 目录下创建 .mdx 文件：

---
title: "文章标题"
description: "文章描述"
pubDate: 2026-04-06
tags: ["标签1", "标签2"]
category: "分类"
---
