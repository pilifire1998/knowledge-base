穿搭推荐（一男一女）

男生穿搭建议：
├─ 上装：2-3件T恤 + 1件薄卫衣 + 1件轻薄防风外套
├─ 下装：2条休闲裤/牛仔裤 + 1条短裤
├─ 鞋子：1双舒适运动鞋（必备！）+ 1双休闲鞋
├─ 配饰：帽子、墨镜、轻便背包
└─ 提醒：日本室内空调较冷，带件薄外套

女生穿搭建议：
├─ 上装：2-3件T恤/衬衫 + 1件针织开衫 + 1件轻薄风衣
├─ 下装：1-2条长裤 + 1条半裙（拍照好看）
├─ 连衣裙：1-2件（京都拍照出片）
├─ 鞋子：1双好走的平底鞋 + 1双小白鞋
├─ 配饰：遮阳帽、小丝巾、小型斜挎包
└─ 提醒：京都寺庙多台阶，避免高跟鞋

通用建议：
├─ 早晚温差约5-8°C，叠穿是王道
├─ 随身带折叠伞（5月偶有阵雨）
├─ 穿着舒适好走的鞋子，日均步数15000+
└─ 京都穿浅色/和风系衣服拍照更出片

免税 & 退税指南

Tax Free 免税条件：
├─ 一般商品：满¥5,000免税（不含税价）
├─ 消耗品：满¥5,000免税（化妆品/食品）
└─ 需出示护照

退税方式：
├─ 百货公司：统一退税柜台（人多需排队）
├─ 药妆店：结账时直接免税
└─ 保留小票，机场可能抽查

⚠️ 注意事项：
├─ 消耗品免税会密封，日本境内不能拆
├─ 建议大商场集中购买，一次退税
└─ 护照随身携带
